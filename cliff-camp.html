An environment map of a canyon wall upon which a secluded camp sits. The map was created in Unreal Engine 4 utilizing custom terrain and premade assets. The goal of this project is to create a virtual environment for a video game cut scene or virtual production while demonstrating my abitlity to create a landscape from scratch.

Environment Packs Used:

“Soul: Cave,” by Epic Games.

“Landscape Backgrounds,” by Gokhan Karadayi.

“Quarry Collections,” by Quixel Megascans.

“M5 VFX Vol2. Fire and Flames,” by JeongukChoi.

“Mediterranean Vegetation: Plant Pack I,” by Project Nature.

“Mediterranean Vegetation: Plant Pack II,” by Project Nature.
